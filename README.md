# ECOM-TEST

- Приложение позволяет осуществлять вывод списка товаров, позволяет отфильтровать их по названию и посмотреть подробную информацию о выбранном товаре в модальном окне. Также функционал приложения позволяет добавить понравившийся товар в избранное или в корзину.
- Использованное API: Моковый эндпоинт 

---


- Вывод карточек товаров полученных из API
- Поиск по названию товара
- Просмотр подробной информации в модальном окне

---

## Дополнительный функционал
- Добавление в избранное: пользователь может добавлять или удалять товары из списка избранных
- Добавление в корзину: пользователь может добавлять или удалять товары из корзины
